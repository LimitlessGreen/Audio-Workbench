# Labeling App — Shortcuts

The table below lists keyboard and interaction shortcuts for the Labeling App.

| Category | Shortcut | Action |
|---|---|---|
| Spectrogram | ++shift+"Drag"++ | Draw a new label region |
| Spectrogram | Double-click | Edit label / open taxonomy search |
| Spectrogram | Shift + Double-click | Reset frequency axis |
| Playback & Navigation | ++space++ | Play / Pause |
| Playback & Navigation | ++arrow-left++ / ++arrow-right++ | Seek backward / forward (fine) |
| Playback & Navigation | ++j++ / ++l++ | Seek backward / forward (coarse) |
| Playback & Navigation | ++home++ | Seek to start |
| Playback & Navigation | ++end++ | Seek to end |
| Labels | ++ctrl+c++ / ++cmd+c++ | Copy selected label |
| Labels | ++ctrl+v++ / ++cmd+v++ | Paste label at current time |
| Labels | ++ctrl+z++ | Undo |
| Labels | ++ctrl+y++ or ++ctrl+shift+z++ | Redo |
| Labels | ++ctrl+d++ | Toggle Stamp mode |
| Labels | ++delete++ / ++backspace++ | Remove selected label |
| Labels | ++g++ | Grab / move focused label |
| Labels | ++1++–++9++ | Apply taxonomy preset to focused label |
| Label Editor | ++arrow-up++ / ++arrow-down++ | Navigate suggestions |
| Label Editor | ++enter++ | Select suggestion / confirm |
| Label Editor | Double-click | Select & immediately confirm |
| Label Editor | ++esc++ | Close editor / cancel |
| Label Editor | ++x++ / ++y++ | Axis lock / constraint when editing or in stamp mode |
| Tags | Double-click | Expand tag editing (Sex, Life Stage, Sound Type, + Tag) |
| Misc | ++middle-button++ + Drag | Pan frequency axis (vertical) |

For additional interactions and shortcuts, open the in-app **Help** panel.
